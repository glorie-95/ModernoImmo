== Description ==

A Wordpress theme for small busineses 

== Setup ==
** Install & Activate the recomended plugins first.
1. create a page and assign it the "home" template under attributes.
2. Go to settings > reading > static page and select the page you just created
3.Install Neptune Real Estate, kirki and ACF plugin (included via TGM)
4. remember to install kirki and ACF (included in TGM) in order to use all the theme's features.
