== Description ==
Construc is a responsive construction WordPress theme best suited for building construction websites.It is an impressive free business WordPress theme for construction, engineering, architecture, real state or building company. The theme is suitable for construction or real state agency and overall construction and real estate sites. It is very important for any company to showcase their service, portfolio, about company, recent updates professionally to let clients be updated and know company service and portfolio.Construction Base is a fully responsive, multi-purpose and SEO friendly theme. The theme has easy and advanced features which lets you design look of the your site without touching code. Some of the important features of theme are more advanced customized option, advanced widgets, About Section, Service Section, Gallery Portfolio Section, Blog/News Section, Social Media Integration, Theme Options, Testimonial, Layout/Design Controls and many others features and options makes it more advanced and useful. Demo Url https://theimran.com/wp/construc/

Quality Design & Standard Coding with HTML5 and CSS3 Bootstrap 4, Easy to use & Customize, Fully Responsive Template, Awesome Sliders (owl-carousel & Slick Slider). fully customizable theme through customizer.

== Frequently Asked Questions ==

= How to Install Theme =

1. In your admin panel, go to Appearance > Themes and click the Add New button and Search for construc, OR
2. If you have theme zip file, click Upload and choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= Front Page Customization ==

1. In the admin area, go to Pages > Add New 
2. Select Front Page template from page attributes.
3. Again In the admin area, go to Settings > Reading
4. In Front page displays choose 'A static page (select below)' and select page for frontpage.

= Front Page Section Customization ==

1. In the admin area, go to appearance > customize > go front page Panel.
2. Here all front page section is available. customize your self.


= How to change content or customize site? =
1. In the admin area, go to Appearance > Customize
2. You wll find different options in customizer
2. Add/Edit Widgets, Page and Post
